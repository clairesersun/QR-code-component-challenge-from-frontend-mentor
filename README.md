### What I learned

I worked on centering items on the screen vertically and horizotally:

```css
dialog {
    background-color: hsl(0, 0%, 100%);
    border-radius: 20px;
    width: 350px;
    margin: auto;
    top: 50%;
    bottom: 50%;
}
```

I also figured out the centering of a footer:
```css
footer {
    text-align: center;
    color: hsl(218, 44%, 22%);
    left: 0;
    bottom: 0;
    width: 100%;
    position: fixed;
}
```
